Transform The Web With Alkaline

Build modern websites in minutes with Alkaline - a low-dependency, framework-less, type-safe Astro starter kit. Featuring 14 unique color themes and more than 2 dozen pre-built Astro components, Alkaline is a great starting point for blogs, portfolios, e-commerce sites, and more.

Configure

Unlock endless possibilities with our intuitive configuration system. Effortlessly customize every aspect of your theme to create a unique and personalized web experience.

Connect

Built on pure Astro - add frameworks only when you need them. Ship less JavaScript by default.

Customize

14 unique themes with consistent design patterns and extensive customization options.

Work Easy

If your package.json looks like a phone book (remember those?) and your build process requires a small server farm, you might be a web developer. Alkaline brings back the simplicity without sacrificing the power, because hard work is overrated.

Fast

Experience blazing speed with Alkaline, optimized for performance and capable of achieving perfect scores on Lighthouse.

Flexible

Leverage Astro's capabilities to build without needing a JavaScript framework. Optionally, add one or multiple frameworks if desired.

Free

What's mine is yours - Alkaline is open-source and MIT licensed.

Created For You

Your tools should be accessible, not intimidating. Whether you're building your first blog or your next startup, Alkaline gives you the foundation to move fast and build things that last.
